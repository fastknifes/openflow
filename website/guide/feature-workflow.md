# Feature 工作流

本文档正在建设中，将介绍 OpenFlow Feature 的完整工作流。

## 工作流概览

Feature 工作流遵循以下步骤：

1. **brainstorm** — 头脑风暴，收集想法与需求
2. **feature** — 创建 Feature 设计，明确目标与约束
3. **writing-plan** — 编写实现计划，拆分任务
4. **implement** — 执行实现
5. **quality-gate** — 质量门禁检查
6. **archive** — 归档完成的 Feature
